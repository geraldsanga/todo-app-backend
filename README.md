# todo-app-backend ⏲️
A django 🐍 powered todo application 📱
all CRUD operations working ✔️

## Things you can do 🤠:
* Create todo 📎
* View todo 👀
* Delete todo ❎

## Project Setup ⚙️

### Install all dependencies 🛒
```
pip install -r requirements.txt
```

### Start the server ⚡

```
python manage.py runserver
```
